# JSON Parser Challenge

A challenge to parse JSON data in JavaScript.

## Instructions

- Implement functions to parse various JSON structures.
- Follow the provided test cases.

## Technologies Used

- JavaScript
- Node.js

## Getting Started

1. Clone the repository.
2. Navigate to the `json-parser-challenge` directory.
3. Run the necessary commands to start the project.
